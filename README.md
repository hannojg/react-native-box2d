### React Native

```sh
yarn add react-native-box2d
cd ios && pod install
```

## Limitations

As the library uses JSI for synchronous native methods access, remote debugging (e.g. with Chrome) is no longer possible. Instead, you should use [Flipper](https://fbflipper.com).
